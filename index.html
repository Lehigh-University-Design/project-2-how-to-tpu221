Step 1: Set Up

Find a flat table with around elbow-distance at the table.
Each time you play each player’s position is a strategic puzzle.

Step 2: The Drop

Each player grabs one token at arm reach and drops when cue is made. After dropping they can’t move pieces—don’t touch the table.

Step 3: Progression

With the table now scattered with tokens, the first player will use one chosen token to aim at the rest using only precision and concentration. Prepare well.

Step 4: Elimination

When someone makes a move that hits a red token or the next best piece, it’s time for elimination. Continue until only one token remains. Every shot counts and the game resets once finished.

Kick Off
  • Because a player is eliminated when their last token rolls off the table, it’s sometimes strategic to use the edge and keep your aim low for better roll control.
  • If you successfully hit a ball off the table and your last stays on, that next person in the progression system advances you.
  • The act of knocking someone off the table counts as a successful turn.
